Quinn Perfetto's API

live @ api.quinnftw.com

An API for the (not so) infamous Quinn Perfetto. Built entirely in nodejs, source can be found on github.

End Points

/fitness - My latests runs recorded from strava
/github - My recent activity on github (commits, pushes, etc.)
/github/followers - People who follow me on github
/github/following - People who I follow on github
/me - Basic information about me pulled from facebook
/me - Basic information about myself
/posts - The most recent posts from quinnftw.com
